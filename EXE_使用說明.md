# PDF 翻譯工具 - EXE 獨立執行版本

## 📦 檔案說明

### 主程式
- **PDF翻譯工具.exe** (約 31.3 MB)
  - 獨立執行檔，無需安裝 Python
  - 包含所有必要的套件和函式庫
  - 支援雙翻譯引擎（Google Translate + Ollama LLM）
  - 雙擊即可直接使用

## 🚀 使用方式

### 方法 1：直接執行
1. 找到 `dist\PDF翻譯工具.exe`
2. 雙擊執行
3. GUI 視窗會自動開啟

### 方法 2：建立桌面捷徑
1. 右鍵點擊 `PDF翻譯工具.exe`
2. 選擇「傳送到」→「桌面（建立捷徑）」
3. 從桌面雙擊捷徑即可使用

### 方法 3：複製到任何地方
- 可以將 `PDF翻譯工具.exe` 複製到任何資料夾
- 可以複製到 USB 隨身碟，在其他電腦上使用
- **不需要安裝任何軟體**

## 📋 系統需求

### 作業系統
- Windows 10 或以上（64位元）
- Windows 7/8 可能也支援（未測試）

### 硬體需求
- RAM: 最少 4GB（建議 8GB 以上，使用 Ollama 時建議 16GB）
- 硬碟空間: 至少 500MB（使用 Ollama 時需額外 5-10GB 用於模型）
- **網路連線**: 
  - Google Translate 引擎：必需
  - Ollama 引擎：不需要（完全離線運作）

## 🎯 使用流程

1. **啟動程式**
   - 雙擊 `PDF翻譯工具.exe`
   - 等待視窗開啟（首次啟動較慢）

2. **選擇檔案**
   - 點擊「瀏覽」選擇要翻譯的 PDF
   - 程式會自動建議輸出檔名

3. **設定選項**
   - 選擇目標語言（預設：繁體中文）
   - 選擇翻譯引擎：
     * **Google Translate** - 快速、需要網路
     * **Ollama LLM** - 需先安裝 Ollama 並下載模型
   - 若選擇 Ollama，可從下拉選單選擇已安裝的模型
   - 可選：設定頁面範圍（例如：1-10）

4. **開始翻譯**
   - 點擊「開始翻譯」按鈕
   - 觀看即時進度
   - 等待完成通知

## ⚠️ 注意事項

### 首次執行
- 首次啟動會較慢（約 10-20 秒）
- 這是正常現象，程式在解壓縮必要檔案
- 後續執行會快很多

### 防毒軟體警告
- 部分防毒軟體可能會誤報
- 這是 PyInstaller 編譯的常見問題
- 檔案完全安全，可放心使用
- 如有疑慮，請檢查原始碼 `pdf_translator_gui.py`

### 網路需求
- **Google Translate 引擎**：
  - 需要網路連線使用 Google Translate API
  - 請確保防火牆允許程式連網
  - 翻譯過程中請保持網路穩定
- **Ollama 引擎**：
  - 完全離線運作，不需要網路連線
  - 需先在本機安裝 Ollama 並下載模型

### 檔案大小
- EXE 檔案約 31.3 MB
- 包含了 Python 直譯器和所有必要套件（PyMuPDF、googletrans、requests）
- 這是獨立執行檔的正常大小

### Ollama 使用說明
如要使用 Ollama 本機翻譯：
1. 下載並安裝 Ollama：https://ollama.ai
2. 下載模型：`ollama pull gemma2:9b` 或其他模型
3. 啟動 Ollama 服務（通常會自動啟動）
4. 在程式中選擇「Ollama LLM」引擎
5. 從下拉選單選擇已安裝的模型
6. 點擊「重新整理」可更新模型列表

## 🔧 疑難排解

### 問題 1：程式無法啟動
**症狀**: 雙擊後沒有反應或閃退

**解決方法**:
1. 確認是 Windows 10/11 64位元系統
2. 嘗試以系統管理員身份執行
3. 檢查防毒軟體是否阻擋

### 問題 2：啟動很慢
**原因**: 首次執行需要解壓縮

**解決方法**:
- 耐心等待 10-20 秒
- 後續執行會快很多
- 可考慮建立桌面捷徑方便使用

### 問題 3：翻譯失敗
**可能原因**:
- 網路連線問題
- Google Translate API 限制

**解決方法**:
1. 檢查網路連線
2. 稍後重試
3. 嘗試分批翻譯（使用頁面範圍）

### 問題 4：中文顯示異常
**解決方法**:
- 檢查原始 PDF 的文字是否可選取
- 圖片中的文字無法翻譯
- 確認輸出 PDF 是否正確生成
